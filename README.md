### Задание 1 Кеширование

Приведите примеры проблем, которые может решить кеширование.

Кеширование — это механизм сохранения часто используемых данных в оперативной памяти для быстрого доступа к ним. Вот основные проблемы, которые решает кеширование:
1. ` Снижение нагрузки на базу данных. При частых запросах к одним и тем же данным кеширование позволяет избежать повторных обращений к БД, что существенно ускоряет работу приложения.`
2. ` Ускорение отклика системы. Кеширование часто используемых данных позволяет получить их из оперативной памяти (которая быстрее диска), что значительно сокращает время ответа системы.`
3. ` Оптимизация производительности. При работе с большими объёмами данных кеширование помогает избежать дорогостоящих операций чтения/записи на диск.`
4. ` Масштабируемость. Кешированные данные могут быть распределены между несколькими серверами, что позволяет обрабатывать больше запросов параллельно.`
5. ` Снижение нагрузки на сеть. Часто используемые данные хранятся локально, что уменьшает сетевой трафик и нагрузку на каналы связи.`
